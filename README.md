OrbitStrike
===========

OrbitStrike is a pixel-style space shooter built with Pyxel. 
Pilot a spaceship, dodge planets, defeat bosses, and rack up points. 
The game features mouse controls, multiple bosses, explosions, and rocket attacks.

Features (version 1.0)
--------
- Pixel-art spaceship
- Moving planets with collision detection
- Boss battles: Galaxy Boss and Pirate Boss
- Standard shooting (left-click) and rocket attacks (right-click)
- Explosions and particle effects
- Score tracking, HP and heat meters
- Moving starfield background

Controls
--------
- Mouse movement: Move Starblade
- Left click: Fire standard weapon
- Right click: Fire rocket (4x damage)

Installation
------------
1. Clone this repository:
   git clone https://github.com/<your-username>/orbitstrike.git
   cd orbitstrike
2. Install Pyxel:
   pip install pyxel
3. Run the game:
   python main.py
